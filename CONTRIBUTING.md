# Contributing

This repository is a research release of **RL-VLA³**, modified from [RLinf](https://github.com/RLinf/RLinf).

For issues related to the upstream infrastructure (installation, simulators, base APIs), please refer to the RLinf documentation and community:

- https://github.com/RLinf/RLinf
- https://rlinf.readthedocs.io/

When contributing changes specific to RL-VLA³ asynchronous VLA training, please keep the scope focused on embodied / VLA workflows under `examples/embodiment/` and the related workers in `rlinf/`.
